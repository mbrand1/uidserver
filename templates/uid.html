Tracking Beacons

Your mobile provider could be tracking you with a unique ID that you cannot remove. If you see a tracking beacon above, that code is being used to track you.

Why are they doing this?

They want to target you with advertising in real time. Selling your information provides cellular carriers with an unlimited revenue stream. Advertisers that are part of the program can query Verizon, AT&T, etc with the code to immediately find out more information about you.

Can I opt out?

No. The code is injected into your browsing session after it leaves your phone and before it reaches a website. It's totally controlled by your cellular carrier. This also means that anyone else that runs a website can view the same code. They might not know what it means, but they can certainly use it to distinguish your session from another. If I own more than one site, I can use it to track you as well.

So what can I do?

The code is only sent during regular unencrypted transmissions. Some ways to get around it include:

  • Encrypt all your traffic / Only use HTTPS connections (not possible with all sites, though). A tool like HTTPS Everywhere can help.
  • Use a VPN service
  • If you use the Chrome Browser, you can proxy your traffic through Google (yes, then Google will see all your traffic instead).
  • Switch providers: Ask a friend on a different provider to hit this site and see if he/she sees any beacons
